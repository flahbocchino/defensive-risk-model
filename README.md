# Defensive Risk Assessment Model

This project presents a simple defensive risk model designed to support protective planning and monitoring posture.

## Overview

The model assigns a normalized score from 0 to 100 based on four core factors:

- Threat
- Exposure
- Vulnerability
- Impact

Formula used:

Rn = (A × E × V × I) ÷ 625 × 100

## Purpose

This is a defensive planning tool intended to help structure risk analysis, prioritize attention, and support protective measures.

It is not a predictive or offensive model.
